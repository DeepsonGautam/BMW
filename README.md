# BMW 3D Website

This is a 3D BMW-themed website I built where the main focus is the car itself, the visuals, and the overall experience.

The site has a BMW car model placed in a 3D environment with dynamic camera angles, smooth movement, and animations that make it feel more interactive instead of just a normal static website.

I also added the BMW logo and a small facts section about BMW — mostly things I personally found interesting while working on the project.

---

## What it is

It’s basically a small 3D experience built around BMW. You can view the car in a more cinematic way with changing perspectives and smooth animations. The goal was to make it feel more like a visual experience rather than a typical webpage.

---

## Features

- 3D BMW car model
- Dynamic camera / perspective movement
- Smooth animations and transitions
- BMW logo integration
- Small facts section about BMW
- Clean 3D scene setup

---

## Built with

- HTML
- CSS
- JavaScript
- Three.js
- GLTF/GLB models

---

## Why I made it

I wanted to build something that feels more alive than a normal website. Cars and 3D stuff are interesting to me, so I combined both and tried to make a simple but immersive experience.

It also helped me understand how 3D scenes, lighting, and camera movement actually work in the browser.
